# Expense Tracker (Java)

## Project Overview
The Expense Tracker is a console-based Java utility application designed to record and analyze daily expenses.  
It allows users to add expenses, categorize them, and view total or category-wise spending.

This project focuses on core Java concepts such as collections, data handling, and menu-driven program design.

---

## Technologies Used
- Java (JDK 8 or above)
- VS Code
- Git & GitHub

---

## Features
- Add daily expenses with category and notes
- View all recorded expenses
- Calculate total expenses
- View category-wise expense summary
- Simple menu-driven console interface
